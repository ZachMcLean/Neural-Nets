Mean-Squared Error¶

Rules:¶

  1. Linear Output Activation Function
  2. Regression Task (Continuous Targets)

Let's start with the derivation of Mean Squared Error (MSE) by assuming that we are learning the mean value of a continuous target variable. The sympy library will be very helpful for this process.

Definition of Normal Distribution and Likelihood function

f(y|μ,σ)=1√2πσ2e−(y−μ)22σ2f(y|μ,σ)=12πσ2e−(y−μ)22σ2

L(μ|y,σ)=1√2πσ2e−(y−μ)22σ2L(μ|y,σ)=12πσ2e−(y−μ)22σ2

In [4]:
norm_dist = sp.Mul((1/sp.sqrt(2*sp.pi*(s**2))),sp.exp((-1*((y-m)**2) / (2*(s**2)))),evaluate=False)
In [5]:

Out[5]:

Note that sympy simplifies the expression a little differently from how we have defined it, but it's still correct.

               xlabel='$y$',ylabel='$f(y|\\mu,\\sigma)$')

Plot likelihood function: this will be... L(μ|y,σ)L(μ|y,σ)

Plot Negative Log-Likelihood: these will be... −log(L(μ|y,σ))−log⁡(L(μ|y,σ)) and −log(L(σ|y,μ))−log⁡(L(σ|y,μ))

Binary Cross-Entropy Loss¶

Rules:¶

  1. Sigmoid Output Activation Function
  2. Binary Classification Task (Discrete, Binary Targets)

Definition of Bernoulli Distribution and Likelihood function

f(y|z)=zy(1−z)(1−y)f(y|z)=zy(1−z)(1−y)

L(z|y)=zy(1−z)(1−y)L(z|y)=zy(1−z)(1−y)

Categorical Cross-Entropy Loss¶

Rules:¶

  1. Sigmoid Output Activation Function
  2. Multiclass Classification Task (Discrete, Integer/One-Hot Targets)

Definition of Multinoulli (Generalized Bernoulli) Distribution and Likelihood function (for three classes - Iris Task)

f(y1,y2,y3|z1,z2,z3)=zy11zy22zy33f(y1,y2,y3|z1,z2,z3)=z1y1z2y2z3y3

L(z1,z2,z3|y1,y2,y3)=zy11zy22zy33L(z1,z2,z3|y1,y2,y3)=z1y1z2y2z3y3

Note: Due to the multidimensional nature of this problem, there are no plots/visualizations provided.
