ViT-B-32 vs. ResNet50

Lecture Date: 2025-04-09

This is a comparison of results for the CIFAR10 task for two architectures: ViT-B-32 and ResNet50. Note that ViT-B-32 does have about three-and-a-half times the number of parameters on this task (88M vs. 24M), but due to the non-overlapping convolution used by ViT-B-32 it can still train on the same batch size as ResNet50. So, the practical training time and memory footprint for ViT-B-32 is competitive even if the number of model parameters is greater. Still, this could be viewed as a biased result just in terms of the number of parameters even if ViT uses those parameters in very different ways from ResNet50.
